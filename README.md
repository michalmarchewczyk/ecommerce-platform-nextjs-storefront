- [React](https://reactjs.org/)
- [Mantine](https://mantine.dev/)

## Installation
```bash
npm install
```

## Running the app in development mode
```bash
npm run dev
```

## Building the app
```bash
npm run build
```

## Starting the app
```bash
npm run start
```
